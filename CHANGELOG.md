## [v0.3.0] - 2026-03-01

### Highlights

- Organization rename from agentplexus to plexusone

### Changed

- **Breaking**: Module path changed from `github.com/agentplexus/structured-evaluation` to `github.com/plexusone/structured-evaluation`
- Updated JSON Schema `$id` URLs to use new module path

### Documentation

- Updated all documentation and examples to use new module path

## [v0.2.0] - 2026-01-26

### Highlights

- Rubric definitions with score anchors for consistent LLM-as-Judge evaluations
- Judge metadata tracking (model, prompt, temperature, tokens)
- Pairwise comparison mode as alternative to absolute scoring
- Multi-judge aggregation with agreement metrics

- Rubric and ScoreAnchor types for defining explicit scoring criteria ([`ba83b8d`](https://github.com/plexusone/structured-evaluation/commit/ba83b8d))

- JudgeMetadata type for tracking LLM model, provider, and prompt ([`4874b1d`](https://github.com/plexusone/structured-evaluation/commit/4874b1d))

## [v0.1.0] - 2026-01-26

### Highlights

- Reusable evaluation framework for LLM-as-Judge and multi-agent workflows
- InfoSec severity levels (Critical, High, Medium, Low) with configurable pass criteria
- DAG-based report aggregation for multi-agent coordination

### Added

- `evaluation` package with EvaluationReport, CategoryScore, Finding, Severity types ([`7a283de`](https://github.com/plexusone/structured-evaluation/commit/7a283de))
- Category scoring with weighted scores (0-10) and pass/warn/fail status ([`0981fed`](https://github.com/plexusone/structured-evaluation/commit/0981fed))
- PassCriteria and Decision types with configurable approval thresholds ([`f32acc9`](https://github.com/plexusone/structured-evaluation/commit/f32acc9))
- EvaluationReport with metadata, categories, findings, and next steps ([`f11ae66`](https://github.com/plexusone/structured-evaluation/commit/f11ae66))
- `summary` package with SummaryReport, TeamSection, TaskResult for GO/NO-GO checks ([`897e09b`](https://github.com/plexusone/structured-evaluation/commit/897e09b))
- `combine` package with DAG-based topological sort using Kahn's algorithm ([`5b384b7`](https://github.com/plexusone/structured-evaluation/commit/5b384b7))
- AggregateResults and AggregateWithDAG for multi-agent coordination ([`5b384b7`](https://github.com/plexusone/structured-evaluation/commit/5b384b7))
- Box-format terminal renderer for summary reports ([`1ea0b48`](https://github.com/plexusone/structured-evaluation/commit/1ea0b48))
- Detailed terminal renderer for evaluation reports with severity grouping ([`1ea0b48`](https://github.com/plexusone/structured-evaluation/commit/1ea0b48))
- JSON Schema generation using invopop/jsonschema ([`2f77e28`](https://github.com/plexusone/structured-evaluation/commit/2f77e28))
- Embedded schemas via `//go:embed` for runtime access ([`2f77e28`](https://github.com/plexusone/structured-evaluation/commit/2f77e28))
